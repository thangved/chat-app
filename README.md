## Install Dependencies

```sh
cd server
yarn
```

```sh
cd app
yarn
```

## Development

- In `./server`

```sh
yarn watch
```

```sh
yarn dev
```

- In `./app`

```sh
yarn start
```

## Production

- In `./server`

```sh
yarn start
```

- In `./app`

```sh
yarn build
```
